# Node.js + MS SQL Server
A simples Node.js + MS SQL Server example.

## How to Run

1. create a MS SQL Server database in your machine or cloud
2. git clone
3. cd nodejs-sqlserver
4. copy .env.example as .env
5. fill .env variables
6. npm start
